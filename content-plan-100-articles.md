2. **SERP Research** → Run 3-5 queries, collect top 15
3. **Competitor Analysis** → Analyze top 5-8 articles
4. **Outline** → Write markdown outline with keyword deployment
5. **User Review** → Wait for approval
6. **Write Article** → Full draft with anti-AI rules
7. **Internal Review** → Sub-agent review against checklist
8. **Keyword Stats** → Count and report deployment
9. **Deliver** → Save to Feishu + local backup
