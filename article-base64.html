什么是 Base64？

Base64 是一种编码方式，它把二进制数据（比如图片、文件）转换成纯文本字符串。为什么需要这个转换？因为很多系统只能安全地处理文本，不能直接传输二进制。

举个例子：你想在 HTML 里直接嵌入一张小图标，而不想让它作为一个单独文件存在。这时候，把图片转成 Base64，然后放进 <img src="data:image/png;base64,..."> 里，就能一张 HTML 文件走天下。

5 个最常见的使用场景

1. 内嵌图片到 HTML/CSS

前端开发中，把小图标转成 Base64 直接写进 CSS，可以减少 HTTP 请求。

2. API 接口里的图片传输

很多 REST API 不方便直接传文件，于是让前端把图片转 Base64，然后放到 JSON 里传过去。

{
  "avatar": "data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEASABIAAD...",
  "username": "moonshot"
}

3. 邮件正文嵌入图片

发 HTML 邮件时，把 logo 和签名图转成 Base64，收件人不需要「下载附件」就能看到完整的邮件。

4. 数据 URI 和 Data URL

浏览器地址栏里可以直接输入 data:text/html;base64,... 来打开一个小页面，完全不依赖服务器。

5. 简单「加密」混淆

Base64 不是加密，但它能把敏感字符串变得不那么一眼可读。比如把一些配置信息编码后存入环境变量。
